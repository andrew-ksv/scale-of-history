## Installation
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: npm install

## Running Tests
- **Run all Cypress tests**: npx cypress run
- **Run a specific Cypress test**: npx cypress run --spec cypress/e2e/frontend/testName.cy.js
- **Run Cypress in UI mode**: npx cypress open
